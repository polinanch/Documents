# Описание класса Salary
Класс для работы с выплатами

## Атрибуты

* Hours : int
* Rub\hour : float
* Teacher : Teacher

## Описание атрибутов

* Hours : int - часы, отработанные преподавателем
* Rub\hour : float - почасовая оплата(рублей в час)
* Teacher : Teacher - преподаватель, которому выплачивается зарплата
